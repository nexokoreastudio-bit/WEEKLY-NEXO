## ✅ 완료된 작업 (2026년 2월)

### 1. 주문 폼 개선 ✅
- **위치**: `index.html` 사이드바
- **추가된 기능**:
  - 유입 경로(`referrer_code`) 필드 추가
  - QR 코드 스캔 시 자동 입력 기능
  - Google Sheets에 유입 경로 저장

**파일 수정**:
- `index.html`: 주문 폼 HTML 추가
- `netlify/functions/form-to-sheets.js`: 유입 경로 필드 처리 추가

**사용 방법**:
1. 사용자가 QR 코드를 스캔하여 사이트 접속
2. URL에 `?ref=STAFF001` 같은 파라미터가 포함됨
3. 주문 폼의 "유입 경로" 필드에 자동 입력됨
4. 주문 제출 시 Google Sheets에 유입 경로가 함께 저장됨

### 2. QR 코드 생성 도구 ✅
- **위치**: `admin.html` → "📱 QR 코드 생성" 탭
- **기능**:
  - 기사님/파트너/회원별 고유 QR 코드 생성
  - 코드 자동 생성 또는 수동 입력
  - QR 코드 이미지 다운로드
  - URL 복사 기능

**파일 생성**:
- `js/qrcode-generator.js`: QR 코드 생성 로직
- `css/admin.css`: 탭 및 QR 코드 스타일 추가
- `admin.html`: QR 코드 생성 탭 추가

**사용 방법**:
1. 관리자 페이지(`admin.html`) 접속
2. "📱 QR 코드 생성" 탭 클릭
3. 소유자 이름, 유형 입력
4. "QR 코드 생성" 버튼 클릭
5. 생성된 QR 코드를 이미지로 다운로드하거나 URL 복사
6. 기사님에게 QR 코드 전달 (명함, 리플릿 등)

**QR 코드 URL 형식**:
```
https://weekly-nexo.netlify.app/?ref=STAFF-홍길동
```

## 🔧 Google Sheets 설정 필요

### 유입 경로 컬럼 추가
Google Sheets에 다음 컬럼을 추가해야 합니다:

1. 기존 주문 폼 시트에 "유입 경로" 컬럼 추가
2. 컬럼 위치: "총 주문 금액" 다음

**컬럼 구조**:
```
제출일시 | 업체명 | 주문자 성함 | 연락처 | 지역 / 설치 환경 | 인치 종류 | 설치 방식 | 구매 수량 | 단가 | 총 주문 금액 | 유입 경로
```

## 📱 QR 코드 활용 방법

### 오프라인 홍보 시나리오
1. **기사님 설치 시**:
   - 기사님이 전자칠판 설치 후 QR 코드 리플릿 전달
   - "이 QR 코드를 스캔하면 넥소 위클리 구독하고 포인트 받을 수 있어요!"

2. **고객이 QR 코드 스캔**:
   - 사이트 접속 (`?ref=STAFF-홍길동` 자동 포함)
   - 주문 폼의 "유입 경로" 필드에 자동 입력됨

3. **주문 제출 시**:
   - Google Sheets에 유입 경로가 기록됨
   - 기사님별 유입 현황 추적 가능
