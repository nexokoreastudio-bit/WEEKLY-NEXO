# NEXO Platform V2.0 개발 가능 여부 빠른 분석

## ✅ 즉시 가능한 작업 (이번 주)

### 1. 회원 수집 폼
- **도구**: Google Forms 또는 Tally
- **시간**: 2-3시간
- **비용**: 무료
- **기능**: 이메일, 이름, 학원명, 유입 경로 수집

### 2. QR 코드 생성 도구
- **도구**: JavaScript 라이브러리 (qrcode.js)
- **시간**: 4-6시간
- **비용**: 무료
- **기능**: 기사님별 고유 코드 생성, 관리자 페이지에 추가

### 3. 주문 폼 개선
- **작업**: 유입 경로 필드 추가
- **시간**: 2-3시간
- **비용**: 무료
- **기능**: Google Sheets에 유입 경로 기록

---

## ⚠️ 백엔드 필요 (1-2개월)

### 1. 회원 시스템
- **필요**: Supabase 도입
- **시간**: 80-120시간
- **비용**: 무료 티어 (초기)
- **기능**: 카카오 로그인, 회원 관리

### 2. 구독자 전용 자료
- **필요**: 인증 시스템
- **시간**: 40-60시간
- **비용**: 무료
- **기능**: 파일 다운로드 인증

### 3. 알림 시스템
- **필요**: 알림톡 API 연동
- **시간**: 40-60시간
- **비용**: 건당 10-20원
- **기능**: 매주 목요일 자동 발송

---

## ❌ 복잡한 기능 (3-6개월)

### 1. 커뮤니티 게시판
- **필요**: DB + 인증 + 파일 업로드
- **시간**: 160-200시간
- **비용**: 무료 티어
- **기능**: 게시글, 댓글, 좋아요

### 2. 포인트 시스템
- **필요**: DB + 트리거 + 로직
- **시간**: 120-160시간
- **비용**: 무료
- **기능**: 적립, 사용, 등급

### 3. 이벤트 시스템
- **필요**: DB + 포인트 시스템
- **시간**: 120-160시간
- **비용**: 무료
- **기능**: 응모, 추첨, 당첨
