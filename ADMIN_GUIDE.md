# 관리자 에디터 사용 가이드

NEXO Weekly 발행물을 쉽게 작성하고 등록할 수 있는 관리자 에디터입니다.

## 🚀 시작하기

### 1. 관리자 페이지 접속

브라우저에서 `admin.html` 파일을 엽니다.

```
http://localhost/admin.html
또는
file:///경로/WEEKLY-NEXO/admin.html
```

### 2. 로그인

- **기본 비밀번호**: `nexo2026`
- 비밀번호는 `js/admin-editor.js` 파일의 `ADMIN_PASSWORD` 변수에서 변경할 수 있습니다.

```javascript
const ADMIN_PASSWORD = 'nexo2026'; // 여기를 변경하세요
```

## 📝 발행물 작성 방법

### 기본 정보 입력

1. **발행일**: 매주 목요일 날짜를 선택하세요 (자동으로 목요일인지 확인)
2. **호수**: 자동 생성되지만 수동으로 변경 가능
3. **제목**: 이번 호의 제목
4. **메인 헤드라인**: 상단에 표시될 큰 제목
5. **서브 헤드라인**: 선택사항
6. **본문 내용**: 이번 호의 주요 내용

### 이미지 등록 (최대 3개)

각 발행물에는 최대 3개의 이미지를 등록할 수 있습니다:

1. **Hero 이미지** (필수): 상단에 표시되는 큰 이미지
2. **Features 이미지 1** (선택): Features 섹션에 표시
3. **Features 이미지 2** (선택): Features 섹션에 표시

#### 이미지 등록 절차:

1. **이미지 파일 선택**: "이미지 선택" 버튼을 클릭하여 파일 선택
2. **미리보기 확인**: 선택한 이미지가 미리보기로 표시됩니다
3. **파일명 확인**: 파일명이 자동으로 입력되지만 수정 가능합니다
4. **정보 입력**:
   - **대체 텍스트**: 이미지 설명 (접근성)
   - **캡션**: 이미지 아래에 표시될 텍스트 (선택사항)

⚠️ **중요**: 이미지 파일을 선택한 후, 반드시 해당 파일을 `assets/images/` 폴더에 복사해야 합니다!

### 업데이트 정보

이번 호의 주요 업데이트를 추가합니다:

- **카테고리**: 소프트웨어, 하드웨어, 서비스, 컨텐츠 중 선택
- **버전**: 예) UMIND v2.3.0
- **날짜**: 업데이트 날짜
- **설명**: 업데이트 내용 설명

"업데이트 추가" 버튼으로 여러 개 추가 가능합니다.

### 업적 정보

넥소의 주요 업적을 추가합니다:

- **타입**: 혁신, 제품 출시, 성장, 협력 중 선택
- **카테고리**: 업적 카테고리
- **날짜**: 업적 달성 날짜
- **제목**: 업적 제목
- **설명**: 상세 설명
- **값**: 업적 값 (예: v2.3.0, 170대 등)
- **이정표**: 이정표 설명

"업적 추가" 버튼으로 여러 개 추가 가능합니다.

### 통계 정보

- **총 설치 대수**: 현재까지 설치된 대수
- **활성 사용자 수**: 활성 사용자 수
- **컨텐츠 업데이트 횟수**: 업데이트 횟수

## ✅ 발행물 등록

모든 정보를 입력한 후 "발행물 등록" 버튼을 클릭하면:

1. **파일 다운로드**: `editions-data.js` 파일이 자동으로 다운로드됩니다
2. **파일 교체**: 다운로드된 파일을 `js/editions-data.js`로 교체하세요
3. **이미지 복사**: 선택한 이미지 파일들을 `assets/images/` 폴더에 복사하세요
4. **확인**: 브라우저를 새로고침하여 새 발행물을 확인하세요

## 🔒 보안

- 관리자 비밀번호는 정기적으로 변경하세요
- `admin.html` 파일은 공개되지 않도록 주의하세요
- 실제 서버 환경에서는 서버 측 인증을 추가하는 것을 권장합니다

## 💡 팁

### 빠른 작성 팁

1. **템플릿 활용**: 이전 발행물을 참고하여 빠르게 작성
2. **이미지 준비**: 발행물 작성 전에 이미지 파일을 미리 준비
3. **날짜 확인**: 반드시 목요일 날짜를 선택하세요

### 이미지 최적화

- 이미지 크기는 적절하게 조정하세요 (권장: 1920px 이하)
- 파일 형식: PNG 또는 JPG
- 파일명은 영문과 숫자, 하이픈(-)만 사용하세요

## 🐛 문제 해결

### 파일이 다운로드되지 않을 때

- 브라우저의 다운로드 설정을 확인하세요
- 팝업 차단을 해제하세요

### 이미지가 표시되지 않을 때

- 이미지 파일이 `assets/images/` 폴더에 있는지 확인
- 파일명이 정확한지 확인
- 브라우저 캐시를 지우고 새로고침

### 발행물이 목록에 나타나지 않을 때

- `js/editions-data.js` 파일이 올바르게 교체되었는지 확인
- 브라우저를 강력 새로고침 (Cmd+Shift+R / Ctrl+Shift+R)
- JavaScript 콘솔에서 오류 확인

## 📞 지원

문제가 발생하면:
1. 브라우저 개발자 도구 콘솔 확인
2. `js/editions-data.js` 파일 형식 확인
3. 이미지 파일 경로 확인
