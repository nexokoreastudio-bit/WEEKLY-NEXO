# 이미지 파일 설정 가이드

## 이미지 파일 복사 방법

제공하신 이미지 파일들을 프로젝트의 `assets/images/` 폴더로 복사해야 합니다.

### 수동 복사 방법

터미널에서 다음 명령어를 실행하세요:

```bash
cd /Users/nexo_jo/Desktop/Nexo_workspace/WEEKLY-NEXO
cp "/Users/nexo_jo/.cursor/projects/Users-nexo_jo-Desktop-Nexo-workspace-WEEKLY-NEXO/assets/"*.png assets/images/
```

또는 Finder에서:
1. `/Users/nexo_jo/.cursor/projects/Users-nexo_jo-Desktop-Nexo-workspace-WEEKLY-NEXO/assets/` 폴더 열기
2. 모든 `.png` 파일 선택
3. `WEEKLY-NEXO/assets/images/` 폴더로 복사
